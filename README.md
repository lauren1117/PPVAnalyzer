## Project Set Up Instructions

1. Check versions:
- Npm version 9.8.1
- Node version 10.14 or above.
- Python version is 3.8.10
2. Install the following:
- pip install flask
- pip install requests
- pip install tensorflow
- pip install pillow
3. Run the project:
- cd frontend
- npm run start:server-dev
- open http://127.0.0.1:5000
